Reglamento Interno

Pérdida de Matrícula

Es un ciclo presencial, por tanto la asistencia es obligatoria.

Si un alumno:

  • Falta 10 días consecutivos desde el inicio de curso se producirá la baja automática del curso.
  • Tiene faltas injustificadas del 15% sobre el total de horas del primer curso se producirá la baja automática del curso.

El curso tiene 600h lo que supone un total de 30 semanas de 20h cada una.

Pérdida de Evaluación

Si un alumno tiene un 15% de faltas injustificadas sobre el total de horas de un módulo concreto perderá el derecho a la evaluación continua, lo que significa que no hará evaluación continua y se examinará en las convocatorias final ordinaria y extraordinaria.

Justificación de Faltas

Se justificarán faltas por:

  • Criterios médicos: Presentar la justificación del médico.
  • Motivos laborales: Imprescindible presentar el contrato laboral donde figure el horario de trabajo.

Contenidos del Curso

Modelos de Inteligencia Artificial

Estudio de los diferentes modelos de IA, sus fundamentos teóricos y aplicaciones prácticas en diversos campos.

Sistemas de Aprendizaje Automático

Desarrollo de sistemas capaces de aprender y mejorar automáticamente a partir de la experiencia sin ser programados explícitamente.

Programación de Inteligencia Artificial

Técnicas y lenguajes de programación específicos para el desarrollo de soluciones basadas en inteligencia artificial.

Sistemas de Big Data

Arquitecturas y tecnologías para el almacenamiento, procesamiento y análisis de grandes volúmenes de datos.

Big Data Aplicado

Aplicaciones prácticas de Big Data en diferentes sectores y resolución de casos de uso reales.

Proyecto Final

Desarrollo de un proyecto integrador que aplique los conocimientos adquiridos en todas las áreas del curso.

Planing del Curso

Planificación general del curso por meses.

Primera Evaluación

Contenidos de cada módulo con ejemplos y ejercicios prácticos:

  • Introducción a los modelos de IA
  • Redes neuronales y aprendizaje profundo
  • Modelos predictivos y clasificación
  • Ejemplo: Implementación de un modelo de recomendación
  • Ejercicio: Creación de un modelo predictivo simple

  • Algoritmos de aprendizaje supervisado y no supervisado
  • Preprocesamiento de datos
  • Evaluación de modelos
  • Ejemplo: Clasificación de imágenes con algoritmos de ML
  • Ejercicio: Implementación de un sistema de clustering

  • Lenguajes para IA (Python, R)
  • Bibliotecas y frameworks (TensorFlow, PyTorch, Scikit-learn)
  • Desarrollo de aplicaciones con IA
  • Ejemplo: Implementación de una red neuronal convolucional
  • Ejercicio: Creación de una API para un modelo de ML

  • Arquitecturas Big Data
  • Procesamiento distribuido (Hadoop, Spark)
  • Bases de datos NoSQL
  • Ejemplo: Implementación de un pipeline de procesamiento de datos
  • Ejercicio: Análisis de grandes volúmenes de datos con Spark

  • Casos de uso en diferentes industrias
  • Visualización de datos
  • Integración de sistemas
  • Ejemplo: Dashboard de análisis de datos en tiempo real
  • Ejercicio: Análisis de un conjunto de datos de una industria específica

Segunda Evaluación

3 proyectos transversales a todos los módulos:

Aula Sensorizada

Diseño e implementación de un sistema de sensores para recopilar datos del entorno del aula y su análisis mediante técnicas de IA y Big Data.

  • Recolección de datos ambientales
  • Análisis de patrones de uso
  • Optimización de recursos
  • Dashboard de monitorización

Reconocimiento de Imágenes

Desarrollo de un sistema de reconocimiento de imágenes utilizando redes neuronales convolucionales y técnicas de visión por computadora.

  • Procesamiento de imágenes
  • Entrenamiento de modelos CNN
  • Clasificación y detección de objetos
  • Aplicación práctica en un caso de uso real

Análisis Industria Musical

Análisis de datos de la industria musical para identificar tendencias, predecir éxitos y recomendar contenido a los usuarios.

  • Procesamiento de audio
  • Análisis de tendencias y patrones
  • Sistemas de recomendación
  • Visualización de resultados
